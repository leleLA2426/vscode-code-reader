# PRD - Code Reader 产品需求文档

## 项目背景

开发者在阅读开源项目或大型代码库时，常常在 IDE 的编辑器、终端、浏览器之间频繁切换。本插件旨在 VSCode 内提供一个专注、高效的代码阅读环境，并通过本地 AI（Ollama）辅助理解复杂代码逻辑。

## 目标用户

- 需要阅读开源项目源码的开发者
- 接手遗留代码库的工程师
- 学习新语言/框架的初学者

## 核心需求

### v0.2.0 — 基础阅读 ✅

**1. 沉浸式代码阅读面板**
- 代码内容渲染 + 行号显示 ✅
- 三种阅读主题（亮色、暗色、护眼褐），设置实时同步 ✅
- 可调节字体大小，设置实时同步 ✅

**2. 文件浏览**
- 侧边栏项目文件树 ✅
- 点击文件在阅读面板中打开 ✅
- 自动折叠 node_modules 等目录 ✅
- 快捷键 Ctrl+Alt+R ✅

**3. 辅助功能**
- 大文件截断 + 提示 ✅
- 中文编码兼容（UTF-8 / GBK） ✅
- 原生模块降级保护 ✅

---

### v0.3.0 — 语法高亮 + 符号导航（计划中）

**4. 语法高亮**
- 基于 tree-sitter 的多语言语法着色（TS/JS/Python/C/C++）
- 关键字、字符串、注释、数字、函数名、类名分色显示
- 三种主题各有适配色值

**5. 代码折叠**
- 基于 tree-sitter AST 的折叠范围计算
- Webview 端折叠/展开按钮
- 默认折叠 import 块和深层嵌套

**6. 符号列表**
- 侧边栏符号树（函数、类、方法、变量）
- 点击符号跳转到对应行
- 支持多层级嵌套展开

---

### v0.3.1 — 智能导航（计划中）

**7. 代码跳转**
- 函数/变量/类的定义跳转
- 引用查找（全项目范围）

**8. 调用关系图**
- 以树形/图形展示函数调用关系
- 标注调用所在文件和行号

---

### v0.4.0 — 搜索与分析（计划中）

**9. 全项目搜索**
- 文件名模糊匹配
- 代码内容 grep（支持正则）
- 符号名搜索

**10. 可视化架构图**
- 基于调用关系生成模块依赖图

---

### v0.5.0 — AI 辅助（计划中）

**11. 代码解释**
- 调用本地 Ollama 模型，流式逐字显示
- 支持追问、多轮对话

**12. 代码摘要**
- 针对整个文件生成摘要

---

### v0.6.0 — 阅读管理（计划中）

**13. 书签** — 代码任意行添加书签（附备注）
**14. 阅读列表** — 待读文件列表
**15. 阅读历史** — 打开记录追踪

---

### v0.7.0 — 导出与笔记（计划中）

**16. 导出** — 代码片段导出 Markdown / PDF
**17. 代码笔记** — 代码块本地标注笔记

## 非功能需求

- **性能**：打开 10k 行文件渲染 < 1s
- **内存**：空闲时 < 100MB
- **稳定性**：Ollama 不可用时扩展不崩溃
- **扩展性**：新增语言支持只需添加 tree-sitter grammar
