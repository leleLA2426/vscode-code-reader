# PRD - Code Reader 产品需求文档

## 项目背景

开发者在阅读开源项目或大型代码库时，常常在 IDE 的编辑器、终端、浏览器之间频繁切换。本插件旨在 VSCode 内提供一个专注、高效的代码阅读环境，并通过本地 AI（Ollama）辅助理解复杂代码逻辑。

## 目标用户

- 需要阅读开源项目源码的开发者
- 接手遗留代码库的工程师
- 学习新语言/框架的初学者

## 核心需求

### P0 — 基础阅读（v0.2.0）

**1. 沉浸式代码阅读面板**
- 多语言语法高亮（至少覆盖 TypeScript, JavaScript, Python, C/C++, Rust, Go, Java）
- 行号显示、代码折叠
- 三种阅读主题（亮色、暗色、护眼褐）
- 可调节字体大小
- 专注模式（隐藏 VSCode 其他 UI 元素）

**2. 文件浏览**
- 侧边栏项目文件树（跟随当前打开的文件夹）
- 点击文件在阅读面板中打开
- 自动折叠 node_modules、.git 等目录

### P1 — 智能导航（v0.3.0）

**3. 符号导航**
- 基于 tree-sitter 提取函数、类、方法、变量、接口
- 侧边栏符号列表，点击跳转
- 当前文件符号大纲

**4. 代码跳转**
- 函数/变量/类的定义跳转
- 引用查找（全项目范围）

**5. 调用关系图**
- 以树形/图形展示函数调用关系
- 标注调用所在文件和行号

### P2 — 搜索与分析（v0.4.0）

**6. 全项目搜索**
- 文件名模糊匹配
- 代码内容 grep（支持正则）
- 符号名搜索
- 搜索结果高亮并支持点击跳转

**7. 可视化架构图**
- 基于调用关系生成模块依赖图
- 在 Webview 中可交互展示

### P3 — AI 辅助（v0.5.0）

**8. 代码解释**
- 选中代码 → 右键 → "解释代码"（快捷键 Ctrl+Shift+E）
- 调用本地 Ollama 模型，流式逐字显示
- 支持追问、多轮对话
- 代码上下文自动注入（文件名、函数名、imports）

**9. 代码摘要**
- 针对整个文件生成摘要
- 在符号列表中附加 AI 生成的注释

### P4 — 阅读管理（v0.6.0）

**10. 书签**
- 在代码任意行添加书签（附备注）
- 侧边栏书签列表，点击跳转
- 支持删除、编辑备注

**11. 阅读列表**
- 将文件添加到「待读」列表
- 支持拖拽排序
- 标记已读/未读

**12. 阅读历史**
- 自动记录打开过的文件
- 按时间倒序排列
- 显示阅读次数

### P5 — 导出与笔记（v0.7.0）

**13. 导出**
- 代码片段导出为 Markdown（保留语法高亮）
- 代码片段导出为 PDF
- 支持附带书签和笔记

**14. 代码笔记**
- 对选中代码块添加本地标记笔记
- 笔记与文件/行号绑定，跨会话持久化

## 非功能需求

- **性能**：打开 10k 行文件渲染 < 1s；搜索 50k 文件项目 < 2s
- **内存**：空闲时 < 100MB，大文件解析峰值 < 500MB
- **稳定性**：Ollama 不可用时扩展不崩溃，功能降级可用
- **扩展性**：新增语言支持只需添加 tree-sitter grammar
